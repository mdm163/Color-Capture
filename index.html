Color Capture

Capture what your image has to offer

Conjecture

Collaging is an artistic way of displaying and organizing images. However, there are currently no tools available that create a collage organized by color gradient. Luckily, a simple program has been developed to solve this problem: Color Capture.

Definition

Color Capture is a simple python program displayed online using an HTML file. The application analyzes the indiviudal pixels of an image and determines the overall percentages of the major colors (red, green, blue, black, and white) present in the image. It has a simple layout that is easy to understand. Currently, the application can only analyze pre-embedded images to show proof of concept; however, future versions will incorporate the option for the user to upload their own images and have them be ranked in color order. A GitHub repository with a README, instructions, and license has been created for future development of the project.

Quality

While this problem isn't on a similar scale to that of climate change, it still is important. There are currently no online applications that provide this type of color gradient analysis. This means that without this application, people would have to organize their photos by hand. When I first moved into my apartment this semester, I attempted to create a collage organized by color gradient. However, it proved to be rather challenging and took days to get it to look somewhat decent (see image above for result). Having an application that automatically analyzes and organizes photos based on color would have saved me much time and aggravation; I believe that creating this application will greatly benefit many in the artistic community.

Policy

Moving forward, Color Capture will transform from a proof of concept to a fully fledged application. First, users will be able to upload their own images to determine color makeup and rank them by color order. Second, users will be able to create accounts and login to access past projects. Third and finally, additional collaging techniques will be incorporated into the website for further applicational use. 3 version updates would be required to accomplish the tasks above. I beleive that implementing the first version update would be the most important step for future use.
